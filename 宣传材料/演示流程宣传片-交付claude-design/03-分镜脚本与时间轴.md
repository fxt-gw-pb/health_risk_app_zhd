# 03 · 分镜脚本与时间轴（≈ 48 秒）

> 布局：**横屏 1920×1080 舞台**。左/右一侧放「竖屏手机」做小程序动态演示，另一侧放**解说面板**（标题 + 要点，即"配音内容上屏"，本片**不做字幕、不强制配音**——所有要讲的话都以面板文字呈现）。
> 手机里的演示是**预先编排好的脚本（演示套壳）**：不跑真实接口，但要把真实的对话/反馈/计算过程演出来。手机内容、解说面板、转场都由**同一条主时间轴 `t`** 驱动（详见 `01-视频创意与规格.md` 的"可逐帧捕获"要求）。
> 机器可读版见 `content/demo-script.json`（手机分步动作）与 `content/narration-panels.json`（解说面板文案）。报告真实数据见 `content/report-data.json`。

演示主角：**李先生，男，58 岁**（数据为冻结医学内核真实计算，请勿改数字）。

## S0 · 开场（0:00–0:03.5）
- **舞台**：深色科技感底（深蓝墨色 + 细网格/光流），中心偏一侧，品牌 Logo（assets/logo.png，蓝青心形＋脉冲）＋ 文字「智评慢病」由小到大、带光晕浮现；竖屏手机从下方/侧边滑入并"亮屏"（屏幕由暗到亮）。
- **解说面板**：
  - 主标题：**智评慢病**
  - 副标题：用对话，看见未来 5 年的慢病风险
- 运动：缓入缓出（cubic-bezier(.22,1,.36,1)），勿过快。

## S1 · 主页：它是什么（0:03.5–0:08.5）
- **手机**：显示首页（屏 A）。镜头从主标题缓慢向下"轻扫"，依次轻微高亮：主标题「读懂未来 5 年的健康风险」→ 三类慢病卡片（糖尿病/高血压/心血管病）→「开始评估」按钮（呼吸光晕）。
- **解说面板**（标题"它是什么"）：
  - 面向普通人的慢病风险自测
  - 评估 **糖尿病 · 高血压 · 心血管病** 未来 5 年风险
  - 全程像聊天，2–3 分钟完成 · 计算在本机
- 结尾：手机上「开始评估」按钮被"点击"（按下回弹 + 流光扫过）→ 进入对话页。

## S2 · 对话问答：怎么用（0:08.5–0:22）
本段是重点，演示**四种输入能力**，节奏明快（每步 1.5–2.2s，气泡逐条入场 + 自动上滚）：

1. 进入对话页，AI 头像 + 自我介绍气泡（可只显示前半句，约 1.4s）：
   > 您好，我是您的健康生活方式助手 🌿 我可以帮您评估慢病风险、解读体检指标、给出生活方式建议……
2. AI 问：`请问您今年多大年纪？（岁）` → 用户在输入框打字 `58` → 发送 → 用户气泡 `58`。（**number 数字题**）
3. AI 问：`您的性别是？` → 底部出现选项 chips `男 / 女` → 点「男」→ 用户气泡 `男`。（**select 选择题**）
4. AI 问：`您平均每天睡几个小时？` → 用户**不点数字，而是打一句大白话**：`晚上一般十二点睡，早上六点半起` → 发送 → 用户气泡显示这句 → 出现绿色确认条：`✓ 已记录 · 每日睡眠时长：6.5小时`。（**自然语言理解** —— 全片最该被看见的能力，给足停留 ≈ 2.4s）
5. AI 问：`请问您的身高（cm）和体重（kg）大约是多少？我来帮您算 BMI ～` → 底部两个输入框填 `174` / `86`，预览条实时跳出 `✨ 自动算出 BMI ≈ 28.4（肥胖）` → 点「完成」→ 用户气泡 `身高 174cm · 体重 86kg` → 确认条 `✓ 已记录 · 身高 174cm、体重 86kg，帮您算得 BMI ≈ 28.4（肥胖）`。（**BMI 自动计算**）
6.（快节奏带过，可各 0.8s）AI 问家族史 `直系亲属里有人得过糖尿病吗？` → 点「有」；问 `您本人被医生确诊过高血压吗？` → 点「**跳过**」（出现灰色"跳过"气泡）。（**一键跳过**）
- **解说面板**（标题"怎么用"，要点随手机步骤逐条点亮）：
  - 选项轻点即答，也能直接打字
  - 听得懂大白话：「十二点睡、六点半起」→ 自动记为 **6.5 小时**
  - 报身高体重，**自动算 BMI**
  - 没把握的题，**一键跳过**（用人群平均值估算）

## S3 · 风险报告：看懂风险（0:22–0:32）
- **手机**：
  1. 出现**阶段小结卡**（`💡 阶段小结 · 基础信息`，定性、无数字，约 1.6s）。
  2. 底部出现两个路径按钮，点击 `直接看风险报告`。
  3. **报告卡入场**（自上而下展开）：报告头一句话总结打出 → 三疾病卡数字**滚动计数**到 `糖尿病 6.6% / 心血管病 21.8% / 高血压 21.4%`（高危卡红、低危卡绿）→ 默认选中**心血管病（高危）**：巨号 `21.8%` + 「高危风险」胶囊 + **风险条填充**到高危区 + Top5 条形**依次生长**（脑卒中家族史↑、腰围↑、睡眠↓、久坐↓、糖尿病家族史↓）。
  4. 点 `❓ 为什么会这样？` → 弹出**为什么卡**（脑卒中家族史↑/腰围↑/… + 一句建议）。
- **解说面板**（标题"看懂风险"）：
  - 三类慢病风险，一屏看清
  - 高 / 中 / 低危分级 + **影响因素 Top 5**
  - 一键追问「为什么」，看清每个因素如何影响你

## S4 · 自由对话：随时追问（0:32–0:39）
- **手机**：报告下方输入框，用户打字提问：`我这种情况，运动应该怎么安排？` → 发送（用户气泡）→ AI 头像呼吸 + TypingDots → **逐字流出**回答（约 3s），文末附**参考来源**。
  - 演示回答（生活方式科普口吻，**不开药、不诊断**）：
    > 建议以中等强度有氧为主，比如快走、骑车、游泳，每周累计 150 分钟以上，分 4–5 次完成；再加 2 次力量练习。久坐时每小时起身活动几分钟。运动循序渐进，如有胸闷气促请停下并就医。
  - 参考来源：`[1] 中国人群身体活动指南（2021）` · `[2] 中国 2 型糖尿病防治指南（2020）`
- **解说面板**（标题"随时追问"）：
  - 像和健康顾问聊天，问饮食、运动、作息
  - 回答**引用权威指南来源**
  - 只做生活方式科普，**不开药、不做诊断**

## S5 · 进一步评估 + 诊断提醒：越测越准（0:39–0:46）
- **手机**：
  1. 点 `➕ 继续补充「血液检查」让评估更准`。
  2. 快节奏填血液指标（蒙太奇式快切，约 2.5s）：收缩压 `136`、空腹血糖 `7.4`、糖化血红蛋白 `6.7`、血脂若干。
  3. 报告**更新**：顶部弹出**就医提醒**（琥珀卡）：
     > ⚠ 就医提醒：您的血糖指标已达到 2 型糖尿病诊断标准（中国 2 型糖尿病防治指南 2020），建议前往医院内分泌科确诊并规范管理。
  4. 糖尿病迷你卡由数字变为 **`疑似已患`**（琥珀），高血压/心血管仍显示更新后的百分比（高血压 15.3% 中危 / 心血管 14.5% 中危）。
- **解说面板**（标题"越测越准"）：
  - 基础信息 → 血液检查 → 血管影像，**三层逐步加深**
  - 指标一旦触及诊断标准，**自动提示就医**

## 节奏与风格备注
- 科技感、表现力强，但**克制不浮夸**：转场用顺滑位移/缩放/流光，少用花哨粒子轰炸；蓝青冷色为主，体现"医疗 + 数据"质感。
- **不要 AI 味的空话**（避免"赋能/智能化/一键开启未来"等套话）。文案已按产品真实语气写好，请直接用。
- 手机内动作要有"被真人操作"的手感：按钮按下回弹、输入有节奏、消息逐条入场并自动上滚。
- 解说面板要点应**与手机当前步骤同步点亮**（手机演到哪个能力，面板对应那一条高亮）。
