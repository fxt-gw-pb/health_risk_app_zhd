  - 已完成层：emerald 浅底 + 绿色 ✓
  - 当前层：青→蓝渐变实心、白字、放大 1.1×、蓝色投影
  - 未到层：slate-100 灰底、灰字
- 最右：重新开始圆形按钮（RotateCcw，仅有消息时出现）

### 2.2 消息气泡
- **AI 气泡**：左侧头像 + 白色气泡（`rounded-2xl rounded-bl-md`，`shadow-premium`，1px slate ring），15px slate-700。入场动画：上移 8px + 淡入 0.28s。

## 3. 底部输入区 Composer（白色玻璃，顶部细分隔线）

> 问诊阶段，输入区上方永远有一颗小药丸：`💬 跳过问答，直接和助手聊聊`（右对齐，灰描边白底胶囊）。

各形态：

**(a) number 数字题**：单行输入框（圆角 2xl，slate-50 底，聚焦变蓝边白底）+ 右侧单位灰字（如 `岁`/`cm`）+「跳过」文字按钮 +「发送」蓝色圆角方钮（44×44，`Send` 图标）。占位符：`输入数值（参考 58）或直接描述`。

**(b) select 选择题**：上排为「选项 chips」——圆角胶囊，2px 蓝色半透边、浅蓝底、`#3B7BEA` 粗体字，点击放大回弹。下方仍有输入框（占位 `或直接打字回答…`）。

**(c) bmi 身高体重**：两个并排输入框（身高 cm / 体重 kg）+ 一条浅蓝预览条 `✨ 自动算出 BMI ≈ 28.4（肥胖）`（Sparkles 图标）+「跳过」+「完成」蓝钮。

**(d) sport 运动**：两个并排输入框（高强度 / 低强度，单位「小时/周」）+ 预览 `✨ 自动换算 ≈ 6.6 MET·h/周` +「跳过」+「完成」。

**(e) choice 选择路径**：竖排大按钮，主选项用蓝渐变实心 + 流光，次选项白底蓝描边；右侧 `ChevronRight`。例：`继续补充「血液检查」指标` / `直接看风险报告`。

**(f) freechat 自由问答**：一颗整宽渐变按钮 `⏱ 开始健康评估，生成我的风险报告` + 下方输入框（占位 `问我健康、饮食、运动方面的问题…`）+ 发送钮。

**(g) postreport 报告后**：整宽渐变按钮 `➕ 继续补充「血液检查」让评估更准` + 输入框（占位 `还有什么健康问题想问我？`）+ 发送钮；下方一颗小胶囊 `↺ 重新评估`。

## 5. 风险报告卡 HealthReport（核心，数据见 `content/report-data.json`）

一张大圆角白卡（`rounded-3xl`），自上而下：

1. **报告头**（蓝→浅蓝渐变，白字）：小标题 `📄 你的健康风险报告` + 一句话总结（17px black）：
   > 您的未来 5 年新发心血管病风险偏高。主要与脑卒中家族史、腰围有关。

2. **就医提醒**（仅诊断触发时出现，琥珀卡 amber-50/200）：`⚠ 就医提醒` + 提示文案（见诊断场景）。

3. **三疾病迷你卡**（一行三列，可点选）：每张含 图标徽章（疾病主题渐变）+ 简称（糖尿病/心血管病/高血压）+ **大号百分数（滚动计数）** + 等级胶囊（低/中/高危）。被选中的卡有 2px 主题描边 + ring。诊断排除时不显示数字，改显 `疑似已患`（琥珀）或 `已诊断`（灰）。
   - 第一层示例：糖尿病 **6.6%** 低危 · 心血管病 **21.8%** 高危 · 高血压 **21.4%** 高危。

4. **选中结局详情卡**（主题浅底 + 描边）：小标题 `5 年新发心血管病` + 巨号百分数（40px black，按等级着色）+ 等级胶囊「高危风险」+ **RiskBar 风险条** + 建议条（白底）+ 蓝色胶囊按钮 `❓ 为什么会这样？`。

5. **影响因素 Top 5**（白卡，`ℹ 影响因素 Top 5`）：5 条水平条形，标签在左（如 脑卒中家族史/腰围/…），条长按贡献绝对值归一；**升高风险=红色渐变 + ↑**，**降低风险=绿色渐变 + ↓**。底部图例：🔴升高风险 🟢降低风险。

6. **模型透明度**（可折叠，slate-50）：`🧪 模型透明度` 展开后含——逐层 C-index 三宫格（当前层高亮）；一段深色代码块（等宽）：
   ```
   Risk(5y) = 1 − S₀^exp(LP)
   S₀ = 0.781734
   LP = -0.0014
   ```
   末行：`房山家系队列 · 随访 2016—2024 · N=3,396 / 事件 1898`。

7. **免责声明**（琥珀淡卡，ShieldAlert）：
   > 本报告基于房山家系队列 Cox 比例风险模型，仅供健康风险参考，**不替代临床诊断**。如有不适请及时就医。

### RiskBar 风险条
水平胶囊条，底色分三段：低危区 emerald-100（左 25%）/ 中危区 amber-100（25%）/ 高危区 rose-100（右 50%）；填充条按等级着色、`width` 0.9s 缓动增长；一个白边深色指示圆点。下方刻度：`10%`(在 25% 处) `20%`(在 50% 处)，两端标 `低危`(绿)、`高危`(红)。展示标尺上限 40%。

## 6. 「为什么」卡 WhyMessage（AI 侧）
